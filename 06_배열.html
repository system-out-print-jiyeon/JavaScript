배열

* 배열이란 ?

자바스크립트에서는 변수선언시 별도의 자료형 지정이 없기 때문에
모든 자료형의 값들을 하나의 배열공간에 담을 수 있다. (자바의 컬렉션과 유사)

* 배열의 선언

배열 선언시 배열의 크기를 지정한채로 선언하거나, 크기 지정안해도 됨 (크기의 제약없음)

* Array 객체의 메소드

1) indexOf(찾고자하는 요소) : 배열에서 요소가 위치해있는 인덱스를 리턴

2) concat(배열, ...) : 여러개의 배열을 결합하고자 할 때 사용

3)join() : 배열에 담긴 값들을 하나의 "문자열"로 결합해서 반환

4) reverse() : 배열에 담긴 값들을 역순으로 바꿔주는 메소드

5) sort() : 배열을 오름차순 정렬시켜주는 메소드

6_1) push(추가할 요소) : 배열의 맨 뒤에 요소 추가하고 배열의 크기 반환
6_1) pop() : 배열의 맨 뒤 요소 제거하고 제거된 요소 반환

7_1) unshift(추가할 요소) : 배열의 맨 앞에 요소 추가
7_2) shift() : 배열의 맨 앞 요소 제거하고 제거된 요소 반환

8_1) slice(시작, 끝) : 배열안의 요소들을 단지 추출해주는 메소드
8_2) splice(시작, 제거수, [추가값]) : 배열의 요소 추출해서 제거 및 추가

9) toString() : 배열에 담긴값들을 하나의 "문자열"로 합쳐서 반환해주는 메소드
